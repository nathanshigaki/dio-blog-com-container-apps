# 🚀 Azure Blog com Container Apps

Este projeto consiste numa aplicação de Blog funcional, empacotada com **Docker** e preparada para ser executada no **Azure Container Apps**. O objetivo é demonstrar o fluxo de implementação de aplicações modernas utilizando containers na nuvem Microsoft Azure, seguindo as práticas de laboratório da **DIO**.

## 📋 Sobre o Projeto

O projeto foca-se na agilidade de escalar aplicações sem gerir infraestruturas complexas (Serverless Containers). A aplicação permite a visualização, detalhe e criação de posts através de uma interface web simples e eficiente.

### Principais Aprendizagens:
* **Containerização:** Criação de imagens leves com Docker.
* **Registry:** Gestão de imagens no Azure Container Registry (ACR).
* **Serverless:** Deploy escalável no Azure Container Apps.
